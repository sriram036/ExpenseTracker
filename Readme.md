# 💸 Expense Tracker (Full Stack)

A simple full-stack application to manage expenses with a **React (Vite + TypeScript)** frontend and a **Spring Boot** backend.

---

## 🧱 Project Structure

```
ExpenseTracker/
│
├── ExpenseTrackerApp/    # React app (Vite + TypeScript)
└── ExpenseTrackerApi/    # Spring Boot API
```

---

## 🚀 Features

* ➕ Add expenses
* 📋 View expenses in a table
* 🔽 Sort by created date
* 📄 Pagination (5 records per page)

---

## ⚙️ Tech Stack

### Frontend

* React (Vite)
* TypeScript
* CSS

### Backend

* Spring Boot
* Java
* REST APIs

---

## 🛠️ Getting Started

### 1.Clone the repository

```
git clone https://github.com/sriram036/ExpenseTracker.git
cd ExpenseTracker
```

---

## ▶️ Run Frontend

```
cd ExpenseTrackerApp
npm install
npm run dev
```

App runs on:

```
http://localhost:5173
```

---

## ▶️ Run Backend

```
cd ExpenseTrackerApi
mvn spring-boot:run
```

API runs on:

```
http://localhost:8080
```
